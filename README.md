# Django-TechProject
Tech project with Django


A basic dynamic website with the Django framework.
This is a mock website for a hotel offering amenities(food & beverage) for guests. The user will be able to select items from a menu to "order".


This website includes:
- Login forms
- Navbar with linked pages
- Footer 
- Dynamic information from the user throughout
- Admin Panel
    - Create, update and delete amenities(food and beverage)
- Database for amenities
